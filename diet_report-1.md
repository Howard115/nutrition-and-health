# 我的飲食分析
我發現我的午餐晚餐飲食習慣就是吃飯配上肉、蛋、豆腐和蔬菜。大部分是吃學餐，大致上還算均衡，不過在飲食種類和一些生活習慣上，我還有可以進步的空間。早餐我通常吃2～3顆水煮蛋，補充蛋白質。

# 優點
1.蛋白質吃得很多元：我有吃到豬肉、雞肉、蛋和豆腐，這樣應該可以補到各種需要的胺基酸。
2.盡量吃原型食物：大部分我吃的都是像米飯、新鮮的肉和蔬菜這種原型食物，比吃一堆加工食品好。
3.每餐都有吃到蔬菜：補充纖維和一些營養素。

# 缺點
1.飲食重複性太高：因為很常吃學餐，變化太少，吃的蔬菜種類也很有限，大多是高麗菜、紅蘿蔔之類的。
2.蔬菜量還是偏少：雖然每餐都有吃，但份量跟飯、肉比起來還是太少了。
3.口味偏重、偏油：有些餐點像是燴飯、滷肉飯，其實都蠻油膩的，而且勾芡和醬汁會讓鈉含量跟熱量偏高。
4.不小心吃到加工食品：這週吃了泡麵、魚板、熱狗，這些東西鈉含量和添加物都偏高。
5.主食太精緻：幾乎餐餐都吃白米飯，很少吃到糙米或其他全穀雜糧。

# 我的改進計畫
1.多吃蔬菜水果：額外用電鍋煮一些蔬菜，補充纖維和一些營養素。
2.飲食多樣化，口味清淡：不要一直吃學餐，偶爾要去市區吃點不一樣的，換換口味也比較均衡。盡量少吃太油的，像滷肉飯、燴飯這種。
